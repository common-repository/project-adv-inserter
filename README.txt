== Description ==

### ITA
Project Adv Inserter è il plugin ufficiale di Projectadv Srl. Si interfaccia con le platform che erogano pubblicità programmatic attraverso la concessionaria pubblicitaria online Project Adv.
Il plugin serve per veicolare annunci pubblicitari nel sito web di destinazione (dove viene installato il plugin), non preleva alcun tipo di dato dal sito web che lo ospita o dai suoi visitatori. Utilizza un file di configurazione remoto, realizzato appositamente per il sito web ed eroga il servizio pubblicitario tramite l'iniezione di file javascript.

Per l'attivazione del servizio contattare [Project Adv](https://www.projectadv.it/), ([Privacy Policy](https://www.projectadv.it/privacy-policy-cookie-e-gdpr-projectadv/))
Il plugin non apporta modifiche al sito web finché il servizio non è attivo.

Il plugin permette la visualizzazione di banner pubblicitari nelle posizioni concordate tra editore e la suddetta azienda. L’esposizione advertising sulle pagine del sito è automatica grazie al download delle configurazioni. Non sono necessari interventi manuali e/o di gestione da parte dell’editore, il plugin agisce in maniera specifica ed efficace sull’impostazione dei banner nei vari formati, sia in versione tabellare che in versione popup. Il funzionamento del plugin è ottimale per visualizzazioni su device desktop, mobile e smart-tv.

Il plugin gestisce il file ads.txt inserendo automaticamente le righe necessarie alla visualizzazione degli advertisement Project Adv e lasciando all'editore la libertà di inserire le proprie righe personalizzate.

Nel caso in cui il sito non disponga di una propria CMP per la gestione del Cookie, il plugin implementa quella di Quantcast, compatibile con le indicazioni legislative e IAB.

***

### ENG
Project Adv Inserter is the official plugin of Projectadv Srl. Project Adv is an advertising agency that helps publishers access programmatic advertising.
The plugin is used to convey advertisements to the target website (where the plugin is installed), it does not collect any type of data from the website that hosts it or from its visitors. It uses a remote configuration file, made specifically for the website and deliver advertisements by injecting javascript files.

To activate the service, contact [Project Adv](https://www.projectadv.it/), ([Privacy Policy](https://www.projectadv.it/privacy-policy-cookie-e-gdpr-projectadv/))
The plugin does not make any changes to the website until the service is active.

The plugin allows the display of advertising banners in the positions agreed between the publisher and Project Adv. The advertising exposure on the website pages is automatic thanks to remote configurations. No manual interventions and/or management by the publisher are required, the plugin handles banners in various formats, both in table and popup version on desktop, mobile and smart-TV devices.

The plugin manages the ads.txt file by automatically inserting necessary lines for displaying Project Adv advertisements and leaving the publisher free to insert his own lines.

If website does not have its own Consent Management Platform (CMP) for cookie management, plugin implements that of Quantcast, compatible with legislative and IAB indications.

== Installation ==

### ITA
Scarica il plugin Project Adv Inserter, inseriscilo nella cartella plugins del tuo sito ed attivalo. Contatta [Project Adv](https://www.projectadv.it/) per attivare il servizio.

***

### ENG
Download Project Adv Inserter plugin, insert it in the plugins folder of your website and activate it. Contact [Project Adv](https://www.projectadv.it/) to activate the service.
